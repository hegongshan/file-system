TensorFlow Model Garden使用TensorFlow实现了许多不同领域的state-of-the-art (SOTA)模型。

### 准备

1.安装TensorFlow Model Garden：

```bash
pip3 install tensorflow tf-models-official
```

2.下载CIFAR-10数据集：

```bash
wget https://www.cs.toronto.edu/~kriz/cifar-10-binary.tar.gz
```
